Slider.js

Slider.js is a light JS library which provides an easy way to create image slideshows.

CSS and Javascript are very separated :

  • The javascript part just does ajax stuff, templating and change a few classes. To perform slide change, only a current class is moved.
  • The CSS part styles the slider and use classes setted by Javascript to perform Slider effects.

For instance, slider.setTransition('transition-left') will set a transition-left class which will be used in CSS to describe the transition with CSS3 Transitions.

Hence, you can make your own transitions very easily. Mixing with CSS3 Transforms, effects are incredible and very efficient (animations are rendered by your browser : accelerated and optimized).

Some code examples

The CSS

Zoom in

/* zoom in */ .slider.transition-zoomin .slide-image { transform: scale(2); } .slider.transition-zoomin .slide-image.current { transform: scale(1); }

Left transition

/* left transition */ .slider.transition-left .slide-image { opacity: 1; left: -100%; } .slider.transition-left .slide-image.currenut { left: 0; } .slider.transition-left .slide-image.current ~ .slide-image { left: 100%; }

Slider

The Slider constructor takes an empty container dom node in argument. var slider = new Slider($('#sliderContainer'));

if DOM is ready - we just need to start the slider engine on it. slider.start();

Fetch a JSON containing photos, create the DOM and start the slider engine. slider.fetchJson("photos_pixar.json");

You already have the JSON, create the DOM and start. slider.setPhotos([ { "src" : "images/monstersinc.jpg", "name": "Monsters Inc" }, { "src" : "images/nemo.jpg", "name": "Nemo" } ]);

Set the slider size. slider.setSize('400px', '300px');

Slide navigation slider.next(); slider.prev(); slider.slide(4);

Themes and transitions are defined in CSS, but you can easily switch theme live with these methods: slider.setTheme('theme-dark'); slider.setTransition('transition-zoomin');

You can chain different methods in any order slider.setSize('400px', '300px') .fetchJson("/photos.json") .slide(5) .setTheme('theme-light') .setTransition('transition-zoomout');

FlickrSlider

Flickr Slider is a subclass of Slider, all Slider method are usable. var slider = new FlickrSlider($('#sliderContainer'));

Default behavior : get recent photos. slider.fetchFlickr();

You can override default flickr options by passing params (see flickr API). slider.fetchFlickr({ per_page: 30, method: 'flickr.photos.search', text: 'China' });
